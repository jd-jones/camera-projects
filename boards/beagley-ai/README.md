# BeagleY-AI

## Overview

Single-board computer based on the TI AM67A processor.

## SSH Access

Hostname: `beagley-ai.local`

```bash
ssh <username>@beagley-ai.local
```

## Flashing

Use [bb-imager](https://github.com/beagleboard/bb-imager) to flash the SD card.

Press `Ctrl+Shift+X` (or `Cmd+Shift+X` on Mac) to open OS Customization settings for SSH keys and WiFi.

Recommended image: Xubuntu (Debian-based default image)
